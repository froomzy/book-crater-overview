Book Crater

Book Crater is a tool that helps you always be getting more books. With a little input from you, lists of books from Book Depository are generated for you each month that allows you to order new books simply. On top of that, Book Crater is a web app designed and built by Dylan Jenkinson to learn new tools and technologies in the web app space.

What Is Book Crater?

Book Crater is two things: A tool for creating orders from Book Depository automatically, so you will always have some new books to look forward to, and a demo project for Dylan Jenkinson to learn new tools and technologies with.

Where Can I Access Book Crater?

There are multiple frontends for Book Crater, built with different technologies. You can use any of these, and have access to your account.
